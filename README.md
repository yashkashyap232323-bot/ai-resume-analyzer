# AI Resume Analyzer 🚀

AI-powered resume analyzer jo score deta hai, skill gaps batata hai aur matching jobs suggest karta hai.

## Setup

1. Files download karo
2. `index.html` VS Code mein open karo
3. **Live Server** extension se run karo
4. API key prompt aayega — Anthropic API key daalo
   - Free key: https://console.anthropic.com

## Files

- `index.html` — Main page
- `style.css`  — Styling
- `app.js`     — AI logic

## API Key Kaise Milegi?

1. https://console.anthropic.com jaao
2. Sign up karo (free)
3. API Keys → Create Key
4. Copy karo aur app mein paste karo

## ⚠️ Note

Production mein API key ko backend (Node.js/Express) pe rakhna — 
kabhi bhi frontend mein hardcode mat karna!
